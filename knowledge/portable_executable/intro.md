# Introduction

Executable files follow a standardized structure called the **Common
Object File Format (COFF)**.

[Portable
Executable](https://learn.microsoft.com/en-us/windows/win32/debug/pe-format)
Files are a COFF formatted executable encapsulating formats such as
Executable (.exe, .cpl, .sys, etc), object code and DLLs.

The PE file format contains integral information required by the Windows
OS to efficiently load and execute the file.
